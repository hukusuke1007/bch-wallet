## Usage for developer

Clone project.

```sh
git clone https://github.com/hukusuke1007/bch-wallet.git
cd bch-wallet
```

Set network in .env.<br>

``` bash
NETWORK = 'testnet'
```

Run commands.

```sh
# Install libraries
$ yarn install

# Run localhost
$ yarn serve
```
